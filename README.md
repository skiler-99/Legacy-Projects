🐦 Flappy Bird (Python + Pygame)

A recreation of the classic Flappy Bird game developed using Python and Pygame.

Features

- Smooth game physics
- Score tracking system
- MySQL leaderboard support
- JSON fallback system
- Asset-based rendering

📁 Location:

/flappybird-python

▶️ Run Locally

pip install -r requirements.txt
python main.py
