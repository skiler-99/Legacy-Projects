# flappybird-python
-📥 **Download Game:** https://github.com/skiler-99/flappybird-python/releases/latest
-Flappy Bird clone in Python (single file) — Pygame + optional MySQL leaderboard.
-Only for learning purpose
## how to run
- pip install -r requirements.txt
- python main2.py
- To store leadboard data create database(player_name, score)
## 🛠️ Technologies Used
- Python
- Pygame
- MySQL (optional leaderboard storage)
## 🛠️ Tech Stack
| Purpose | Technology |
|---------|-------------|
| Game development | Python + Pygame |
| Database storage (optional) | MySQL |
| Offline storage fallback | JSON File |
